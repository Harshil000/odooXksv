1. PROJECT OVERVIEW
================================================================================

VendorBridge is a comprehensive Procurement & Vendor Management ERP system with 
10 key screens managing the complete procurement workflow from RFQ creation 
through vendor quotation submission, comparison, approval, purchase order 
generation, and invoice management.

TECH STACK:
-----------
✅ Frontend: React 19, Vite 8.0.12, React Router 7
✅ Styling: SCSS with dark theme & glass morphism effects
✅ State Management: Context API + Custom Hooks
✅ HTTP Client: Axios with baseURL configuration
✅ Notifications: React-toastify
✅ Build Tool: Vite with HMR (Hot Module Replacement)
✅ Authentication: Cookie-based with JWT tokens

DESIGN SYSTEM:
--------------
• Color Scheme: Dark theme (#0f172a, #1a1f2e)
• Accent Colors: Purple/Pink gradient (#a78bfa, #c084fc, #6366f1)
• Text Primary: #e2e8f0 / #cbd5e1
• Text Secondary: #94a3b8 / #64748b
• Effects: Glass morphism (backdrop-filter: blur), smooth animations
• Responsive: Mobile-first with breakpoints at 768px and 480px

USER ROLES:
-----------
1. Procurement Officer - Creates RFQs, manages vendors
2. Vendor - Submits quotations against RFQs
3. Manager/Approver - Approves/rejects quotations and POs
4. Admin - Overall system management and reports
